#Sprint Week-3 Project: Spots

An interactive image posting and sharing website.

##Description

This is an image sharing based webpage similar to a social media platform. Display has been optimized to function on different screens. A simple and sleek display for sharing important content.

##Tech Stack

-HTML
-CSS
-Responsive Design: add button, like button, delete button, cancel button & edit button using Java Script
-design based on: [Link to the project on Figma] (https://www.figma.com/design/mXGZ6wZ4QPKx5KjpHX9QCV/Sprint-9-Project--Spots?node-id=2601-356&t=1mZMnTc4XUT2R6Y5-0)

-Javascript: form validation, editing the profile, and add new posts to the console.

-display size based CSS features: @media in CSS

-Website uses Api.js to link to server for better user experience and logging of changes made to the webpage.
